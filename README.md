# Java API Gateway Demo

A hands-on project demonstrating how to build and integrate an API Gateway using Spring Cloud Gateway and Eureka in a microservices architecture.

> **Project Status:** 🚧 Repository Initialization

## Overview

This project is being developed as a learning exercise to understand how API Gateways work in real-world Java microservices applications.

The project will progressively introduce:

- Spring Cloud Gateway
- Netflix Eureka Service Discovery
- Spring Boot Microservices
- Service-to-Service Communication
- Routing
- Load Balancing
- Unit Testing
- Integration Testing

## Technologies

- Java 25
- Spring Boot
- Spring Cloud Gateway
- Netflix Eureka
- Maven
- JUnit 5
- Mockito

## Learning Objectives

- Understand microservices architecture.
- Learn the purpose of an API Gateway.
- Explore service discovery using Eureka.
- Implement request routing between services.
- Apply testing best practices.
