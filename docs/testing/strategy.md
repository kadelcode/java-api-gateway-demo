# Testing Strategy

Testing is an essential part of this project.

## Unit Testing

Business logic will be tested using:

- JUnit 5
- Mockito

## Controller Testing

REST endpoints will be tested using:

- MockMvc

## Integration Testing

Spring Boot Test will be used to verify:

- Context loading
- Endpoint integration
- Gateway routing

## Test Coverage Goals

Every service should include:

- Service tests
- Controller tests
- Integration tests
